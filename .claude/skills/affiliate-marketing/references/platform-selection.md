# Platform Selection

## Platform Comparison Matrix

| Platform | Best For | Price | Key Features |
|----------|----------|-------|--------------|
| FirstPromoter | Early SaaS, Stripe users | $49/mo | Stripe/Paddle native, email automation, 18+ metrics |
| Rewardful | Stripe-first startups | $29/mo | Simple setup, fraud protection, tracking |
| PartnerStack | Enterprise, complex ecosystems | $500+/mo | Marketplace, multi-program, enterprise reporting |
| Impact.com | Large B2C, influencers | Custom | Real-time compliance, bot filters |
| Tapfiliate | Customizable programs | $59/mo | White-label, custom fraud rules |

## FirstPromoter (Recommended for Early-Stage)

**Best For:** SaaS <$5M ARR, Stripe/Paddle billing

**Pros:**
- Purpose-built for SaaS recurring
- Stripe/Paddle/Chargebee integration
- Email automation (unique feature)
- Simple setup, low friction
- Fraud protection built-in
- $49/mo starter

**Cons:**
- No partner marketplace
- 1,000 affiliate limit on Starter
- Manual recruitment required

## PartnerStack (Enterprise)

**Best For:** SaaS >$5M ARR, complex partner ecosystems

**Pros:**
- Industry-leading for B2B partnerships
- Built-in partner marketplace (passive discovery)
- Multiple concurrent programs
- Enterprise-grade reporting

**Cons:**
- $500+/month pricing
- Steep learning curve
- Requires onboarding support

## Rewardful

**Best For:** Stripe-first, bootstrapped SaaS

**Pros:**
- Stripe-native integration
- Simple pricing ($29/mo)
- Quick setup (30 minutes)
- AI fraud detection

**Cons:**
- Stripe-focused
- Fewer advanced features

## Integration Checklist

- [ ] Payment processor (Stripe/Paddle/Chargebee)
- [ ] CRM (HubSpot, Salesforce)
- [ ] Analytics (GA, Mixpanel)
- [ ] Email (Mailchimp, SendGrid)
- [ ] Webhook support
- [ ] API access

## Stage-Based Recommendations

```
Bootstrapped (<$100K ARR):
→ Rewardful $29/mo

Early-Stage ($100K-$1M ARR):
→ FirstPromoter $49/mo

Growth ($1M-$5M ARR):
→ FirstPromoter Business $99/mo

Scale ($5M+ ARR):
→ PartnerStack or Impact.com
```
