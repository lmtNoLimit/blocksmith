# Blog Post Templates

## Template Types

### 1. How-To Guide
```markdown
# How to [Achieve Outcome]

## Introduction
[Problem statement + what they'll learn]

## Prerequisites
- [Requirement 1]
- [Requirement 2]

## Step 1: [Action]
[Detailed instructions]

## Step 2: [Action]
[Detailed instructions]

## Step 3: [Action]
[Detailed instructions]

## Common Mistakes to Avoid
[List of pitfalls]

## Conclusion
[Summary + next steps]
```

### 2. Listicle (Top X)
```markdown
# [Number] [Best/Top/Essential] [Things] for [Audience/Purpose]

## Introduction
[Why this list matters]

## 1. [Item]
[Description + why it's included]

## 2. [Item]
...

## Conclusion
[Summary + CTA]
```

### 3. Ultimate Guide
```markdown
# The Ultimate Guide to [Topic]

## Table of Contents
[Linked sections]

## What is [Topic]?
[Definition + context]

## Why [Topic] Matters
[Benefits + statistics]

## [Main Section 1]
[Comprehensive coverage]

## [Main Section 2]
...

## FAQ
[Common questions]

## Resources
[Tools, templates, further reading]
```

### 4. Comparison Post
```markdown
# [Option A] vs [Option B]: [Outcome They Want]

## Quick Summary
| Feature | Option A | Option B |
|---------|----------|----------|

## [Option A] Overview
[Features, pros, cons]

## [Option B] Overview
[Features, pros, cons]

## Which Should You Choose?
[Decision criteria]
```

### 5. Case Study
```markdown
# How [Company] Achieved [Result] Using [Method]

## The Challenge
[Problem they faced]

## The Solution
[What they implemented]

## The Results
[Metrics and outcomes]

## Key Takeaways
[Lessons learned]
```
