# Editorial Calendar Template

## Required Fields

| Field | Description |
|-------|-------------|
| Publish Date | Scheduled date |
| Content Type | Blog, video, social, email |
| Topic/Title | Working title |
| Target Keyword | Primary SEO keyword |
| Author/Owner | Content creator |
| Status | Draft/Review/Scheduled/Published |
| Distribution | Channels for promotion |

## Monthly Calendar View

```markdown
## [Month Year] Editorial Calendar

### Week 1 (Date - Date)
| Date | Type | Title | Keyword | Owner | Status |
|------|------|-------|---------|-------|--------|
| Mon | Blog | [Title] | [kw] | [Name] | Draft |
| Wed | Email | [Title] | - | [Name] | Review |
| Fri | Social | [Title] | - | [Name] | Ready |

### Week 2...
```

## Content Planning Process

1. **Quarterly Planning**
   - Review performance data
   - Identify themes and campaigns
   - Set content goals

2. **Monthly Planning**
   - Map content to calendar
   - Assign owners
   - Schedule production

3. **Weekly Planning**
   - Review upcoming content
   - Coordinate with team
   - Adjust as needed

## Status Workflow

```
Idea → Brief → Draft → Edit → Review → Approved → Scheduled → Published
```
