# Creativity - Creative Direction Guide

Quick reference for creative direction in digital marketing campaigns.

## Core Principle

**92% prefer authentic content over polished ads.** Prioritize real, relatable content over perfection.

## Quick Style Selection

Match style to brand + audience:
- **Premium/Luxury**: Minimalist Clean, Quiet Luxury, Dark Moody Cinematic
- **Tech/SaaS**: Futuristic Sci-Fi, Gradient Smooth, Bold Geometric
- **Youth/Gen Z**: Raw Unpolished, Humor, Nostalgic 90s, Neon Cyber
- **Wellness/Organic**: Earthy Natural, Emotional Storytelling, Pastel Soft
- **B2B/Finance**: Authoritative Expert, Data Visualization, Formal Professional
- **Community**: User-Generated, Conversational Casual, Behind-the-Scenes

## Creative Brief Essentials

1. **Objective** - What action/perception change
2. **Audience** - Demographics, psychographics, platform behavior
3. **Key Message** - Single memorable takeaway
4. **Tone** - From style reference table
5. **Constraints** - Budget, compliance, brand guardrails
6. **Success Metrics** - CTR, engagement, conversion targets

## Platform Quick Specs

| Platform | Ratio | Length | Style Note |
|----------|-------|--------|------------|
| TikTok | 9:16 | 15-60s | Raw, trending sounds, hook in 3s |
| Instagram Reels | 9:16 | 15-90s | Polished-authentic, music-driven |
| YouTube | 16:9 | 30s-10m | Story arc, high production value |
| LinkedIn | 1:1, 4:5 | 30s-2m | Professional, value-first |
| Stories | 9:16 | 15s | Immediate hook, swipe CTA |

## Key Stats

- Humor increases recall 33% (only 33% of ads use it)
- Color drives 85% purchase decisions
- Brand recognition +80% with consistent colors
- UGC: 28% higher engagement, 4x CTR, 50% lower CPA

## Implementation Checklist

Pre-production:
- [ ] Brand guidelines defined
- [ ] Audience persona clear
- [ ] Creative style selected from reference table
- [ ] Platform specs identified
- [ ] Success metrics set

Production:
- [ ] Design for sound-off first (captions)
- [ ] Hook in first 3 seconds
- [ ] Mobile-first (80%+ mobile viewing)
- [ ] Include diverse representation
- [ ] Build modular assets for testing

Post:
- [ ] A/B test one variable at a time
- [ ] Monitor 48-72h before pivoting
- [ ] Document learnings
- [ ] Refresh before ad fatigue (2-4 weeks)
