# KPI Tracking Guide

## Core Metrics

### Engagement
| Metric | Formula | Target |
|--------|---------|--------|
| DAU/MAU Ratio | Daily Active / Monthly Active | >20% |
| Session Length | Avg time per session | +15% |
| Return Rate | Users returning within 7 days | >40% |

### Gamification-Specific
| Metric | Target | Frequency |
|--------|--------|-----------|
| Points Awarded/Day | Trend up | Daily |
| Badge Earn Rate | 5-15% | Weekly |
| Challenge Completion | >40% | Weekly |
| Streak Maintenance | >30% | Daily |
| Leaderboard Views | >25% | Daily |

### Retention
| Metric | Target |
|--------|--------|
| D1 Retention | >50% |
| D7 Retention | >25% |
| D30 Retention | >15% |
| Churn Rate | <5%/mo |

### Revenue Impact
| Metric | Target |
|--------|--------|
| ARPU | +20% vs baseline |
| Conversion | +15% vs control |
| LTV | +25% gamified |
| Redemption Rate | 60-80% |

## Alert Thresholds

| Metric | Warning | Critical |
|--------|---------|----------|
| DAU Drop | -10% DoD | -25% DoD |
| Streak Breaks | +20% vs avg | +50% |
| Challenge Abandon | >60% | >80% |
| Churn Spike | +2% monthly | +5% |

## Reporting Cadence

- **Daily:** Points, streaks, challenges, anomalies
- **Weekly:** Engagement lift, retention, A/B results
- **Monthly:** Revenue impact, LTV, tier distribution
- **Quarterly:** ROI calculation, strategic review

See [analytics-events.md](analytics-events.md) for event schemas.
