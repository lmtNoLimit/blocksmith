## Detailed Mechanics

### Points System
- **Earning:** 10 pts/$1 spend, 50 pts/referral, 5 pts/engagement
- **Redemption:** 100 pts = $1 value (10% return rate)
- **Multipliers:** 2x weekends, 3x birthdays, bonus challenges
- **Expiry:** 12-month rolling (creates urgency)

### Badges
- **Tiers:** Common → Rare → Epic → Legendary
- **Categories:** Achievement, milestone, social, seasonal
- **Display:** Profile showcase, 3-badge highlight
- **Unlock rate target:** 5-15% per badge type

### Leaderboards
- **Types:** Daily, weekly, monthly, all-time, friends-only
- **Display:** Top 10 visible, personal rank always shown
- **Reset:** Weekly resets drive re-engagement
- **Segments:** By tier, region, or player type

### Streaks
- **Milestones:** 3, 7, 14, 30, 60, 100, 365 days
- **Rewards:** Exponential (day 7 = 7x points, day 30 = 50x)
- **Protection:** Streak freeze (earned or purchased)
- **Recovery:** 24-48hr grace period option

### Challenges
- **Duration:** Daily (1-day), Weekly (7-day), Monthly
- **Difficulty:** Easy (80% success), Medium (50%), Hard (20%), Expert (5%)
- **Types:** Individual, team, competitive, collaborative
- **Rewards:** Points + badges + exclusive unlocks

### Progress Bars
- **Visibility:** Always visible for active goals
- **Psychology:** Show % complete, remaining steps
- **Momentum:** Start at 10-20% (endowed progress)
- **Completion:** Celebration animation + reward

## Combination Patterns

### Loyalty Program Stack
Points + Tiers + Streaks + Challenges
```
User earns points → Reaches tier → Maintains streak → Completes challenges → Earns multipliers
```

### Referral Program Stack
Leaderboard + Dual Rewards + Social Proof + Scarcity
```
User refers → Both rewarded → Shown on leaderboard → Limited-time bonus
```

### Onboarding Stack
Progress Bar + Micro-Badges + Unlockables + Email Drips
```
User signs up → Sees progress → Earns badges → Unlocks features → Gets encouragement
```

## Implementation Priority

**Week 1-2:** Points + Progress Bar (foundation)
**Week 3-4:** Badges + Challenges (engagement)
**Week 5-6:** Streaks + Leaderboards (retention)
**Week 7-8:** Variable Rewards + Unlockables (delight)
