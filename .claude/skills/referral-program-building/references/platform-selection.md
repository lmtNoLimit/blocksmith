# Referral Platform Selection

## Quick Decision Tree

```
SaaS + Stripe/Paddle? → Rewardful
Ecommerce + Shopify? → ReferralCandy
Custom campaigns/contests? → Viral Loops
Recurring commissions? → FirstPromoter
Enterprise + custom rules? → Voucherify
Full control needed? → Build custom
```

## Platform Comparison

### Rewardful
- **Best for:** SaaS subscriptions
- **Price:** $49-500/mo
- **Setup:** 5-10 minutes, no coding
- **Integrations:** Stripe, Paddle, Gumroad
- **Strengths:** Simplest SaaS setup, affiliate+referral blend
- **Weaknesses:** Limited customization
- **Users:** ConvertKit, Segment

### ReferralCandy
- **Best for:** Ecommerce/physical products
- **Price:** $49-3999/mo
- **Setup:** 1-click native plugins
- **Integrations:** Shopify, WooCommerce, BigCommerce
- **Strengths:** Long-established, proven at scale
- **Weaknesses:** Less flexible for SaaS, legacy feel

### Viral Loops
- **Best for:** Custom campaigns, growth teams
- **Price:** Custom pricing (~$500+/mo)
- **Setup:** Visual builder, 15+ templates
- **Integrations:** Shopify, Webflow, WordPress, email platforms
- **Strengths:** Flexibility, contests, leaderboards, gamification
- **Weaknesses:** Steeper learning curve

### FirstPromoter
- **Best for:** Subscription businesses
- **Price:** Custom pricing
- **Setup:** Dashboard-based configuration
- **Integrations:** Multiple payment platforms
- **Strengths:** Recurring commission tracking, native email
- **Weaknesses:** More complex than Rewardful

### Voucherify
- **Best for:** Enterprise, API-first
- **Price:** Custom (enterprise)
- **Setup:** API integration, SDKs available
- **Integrations:** 100+ integrations
- **Strengths:** Maximum flexibility, custom rules, fraud detection
- **Weaknesses:** Requires technical implementation

## Build vs Buy Decision

**Build Custom When:**
- Unique business logic requirements
- Deep payment integration needed
- White-label experience required
- Have 3-4 weeks engineering capacity
- Budget: $20-50k development

**Buy Platform When:**
- Standard referral mechanics
- Time-to-market priority
- Limited engineering resources
- Budget: $50-500/mo

## Integration Checklist

- [ ] Payment provider compatibility (Stripe, Paddle, etc.)
- [ ] Existing tech stack integration
- [ ] Email platform connection
- [ ] Analytics/tracking setup
- [ ] Fraud detection capabilities
- [ ] Dashboard/reporting needs
- [ ] Mobile app support (if applicable)
