# On-Page SEO Checklist 2025

## Title & Meta Elements (6 items)

### [ ] Title Tag Optimization
- Length: 50-60 characters (< 580 pixels)
- Primary keyword in first 50 characters
- Brand name at end (if space permits)
- Unique title per page
- **Pass:** Contains primary keyword, under 60 chars
- **Fail:** Generic, duplicate, missing keyword, >60 chars

### [ ] Meta Description
- Length: 150-160 characters (< 920 pixels)
- Includes primary keyword and CTA
- Compelling, action-oriented copy
- Unique per page
- **Pass:** 150-160 chars, keyword present, CTA included
- **Fail:** Missing, duplicate, >160 chars, no CTA

### [ ] H1 Header Tag
- One H1 per page
- Contains primary keyword
- Different from title tag
- Under 70 characters
- **Pass:** Single H1, keyword included, unique
- **Fail:** Multiple H1s, no keyword, identical to title

### [ ] URL Slug Structure
- Lowercase, hyphen-separated
- Contains primary keyword
- Under 75 characters
- No stop words (if, and, the, etc.)
- **Pass:** Clean, keyword-rich, short
- **Fail:** Long, parameter-heavy, no keywords

### [ ] Canonical Tag
- Self-referencing canonical on original content
- Points to preferred version if duplicates exist
- HTTPS version specified
- **Pass:** Correct canonical URL set
- **Fail:** Missing, incorrect, or conflicting canonical

### [ ] Meta Robots Tag
- Index,follow for public pages
- Noindex for thin/duplicate content
- Nofollow for untrusted links (if applicable)
- **Pass:** Appropriate directive set
- **Fail:** Unintentional noindex on important pages

## Content Optimization (11 items)

### [ ] Primary Keyword Placement
- In first 100 words
- In H1, H2 tags
- In title and meta description
- Natural density (0.5-2%)
- **Pass:** Keyword appears in all key locations naturally
- **Fail:** Keyword stuffing or completely absent

### [ ] Keyword Variations & LSI
- 5-10 semantic keyword variations
- Related entity mentions
- Synonym usage
- **Pass:** Natural semantic coverage
- **Fail:** Exact match repetition only

### [ ] Content Length
- Blog posts: 1,500-2,500 words
- Pillar pages: 2,500-4,000 words
- Product pages: 500-1,000 words
- **Pass:** Meets minimum for content type
- **Fail:** Thin content (<300 words)

### [ ] Heading Hierarchy (H2-H6)
- Logical structure (H2 → H3 → H4)
- Keywords in subheadings
- Min 4-6 H2 tags per long-form content
- **Pass:** Proper hierarchy, keyword-rich
- **Fail:** Skipped levels, generic headings

### [ ] Content Freshness
- Publication date visible
- Last updated date shown
- Regular content updates (quarterly for evergreen)
- **Pass:** Current year references, updated data
- **Fail:** Outdated stats, broken references

### [ ] E-E-A-T Signals
- Author bio with credentials
- Citations and references
- Expert quotes or contributions
- Original research or data
- **Pass:** Clear expertise demonstrated
- **Fail:** No author info, uncited claims

### [ ] Content Uniqueness
- 100% original content (no scraping)
- Unique perspective or value-add
- Not duplicated across site
- **Pass:** Passes plagiarism check, unique insights
- **Fail:** Duplicate, thin, or scraped content

### [ ] Paragraph Structure
- Max 3-4 sentences per paragraph
- White space between sections
- Scannable format
- **Pass:** Easy to scan, broken into chunks
- **Fail:** Large text blocks, no breaks

### [ ] Sentence Readability
- Vary sentence length (8-25 words)
- Active voice (>80% of sentences)
- Simple vocabulary (avoid jargon)
- **Pass:** Flesch Reading Ease 60-70+
- **Fail:** Complex sentences, passive voice heavy

### [ ] Bullet Points & Lists
- Use for 3+ related items
- Numbered lists for sequential steps
- Bulleted lists for non-sequential
- **Pass:** Lists used appropriately
- **Fail:** Large paragraphs that should be lists

### [ ] Call-to-Action (CTA)
- Clear CTA above fold and at end
- Relevant to content topic
- Action-oriented language
- **Pass:** Visible CTA with clear next step
- **Fail:** No CTA or buried CTA

## Technical Elements (10 items)

### [ ] Page Load Speed
- Core Web Vitals passing
- LCP <2.5s, FID <100ms, CLS <0.1
- Mobile speed score >85
- **Pass:** All Core Web Vitals green
- **Fail:** Any metric in red zone

### [ ] Mobile Responsiveness
- Mobile-friendly test passed
- Readable font sizes (16px min)
- Touch-friendly buttons (48px min)
- **Pass:** Google Mobile-Friendly Test passes
- **Fail:** Horizontal scroll, tiny text

### [ ] HTTPS Security
- SSL certificate active
- All resources loaded via HTTPS
- No mixed content warnings
- **Pass:** Green padlock, HTTPS everywhere
- **Fail:** "Not secure" warning

### [ ] Structured Data (Schema)
- Appropriate schema type implemented
- Article, Product, FAQ, HowTo, etc.
- Valid JSON-LD format
- **Pass:** Rich results test passes
- **Fail:** Errors in schema validator

### [ ] Open Graph Tags
- og:title, og:description, og:image
- og:type appropriate for content
- Twitter card tags
- **Pass:** All required OG tags present
- **Fail:** Missing social meta tags

### [ ] Image Optimization
- File size <200KB per image
- WebP or next-gen format
- Compressed without quality loss
- **Pass:** Images load fast, modern format
- **Fail:** Oversized files, slow loading

### [ ] Lazy Loading
- Images lazy load below fold
- Iframe lazy loading
- Native loading="lazy" attribute
- **Pass:** Below-fold images deferred
- **Fail:** All images load on page load

### [ ] Breadcrumb Navigation
- Breadcrumbs on all non-home pages
- Schema markup for breadcrumbs
- Clickable links in path
- **Pass:** Breadcrumbs visible with schema
- **Fail:** No breadcrumbs on deep pages

### [ ] 404 Error Handling
- Custom 404 page with helpful links
- Search functionality on 404
- Link to homepage/popular pages
- **Pass:** Useful 404 with navigation
- **Fail:** Default server 404 page

### [ ] XML Sitemap Inclusion
- Page included in XML sitemap
- Priority and changefreq set
- Submitted to Search Console
- **Pass:** URL in sitemap, indexed
- **Fail:** Missing from sitemap

## Internal & External Links (6 items)

### [ ] Internal Link Count
- Min 3-5 internal links per page
- Links to related content
- Varied anchor text
- **Pass:** 3+ contextual internal links
- **Fail:** No internal links or orphan page

### [ ] Internal Anchor Text
- Descriptive, keyword-relevant
- Avoid "click here" or "read more"
- Natural in context
- **Pass:** Descriptive anchor text used
- **Fail:** Generic anchor text

### [ ] External Link Quality
- 1-3 external links to authoritative sources
- Links to .edu, .gov, industry leaders
- Relevant to topic
- **Pass:** Quality outbound links present
- **Fail:** No external links or spammy links

### [ ] Link Destination Validity
- All links return 200 status
- No broken links (404s)
- No redirect chains (>2 hops)
- **Pass:** All links functional
- **Fail:** Broken or redirected links

### [ ] Nofollow Attributes
- Affiliate links have rel="nofollow" or "sponsored"
- UGC links have rel="ugc"
- Paid links have rel="sponsored"
- **Pass:** Proper rel attributes on paid/UGC links
- **Fail:** Missing nofollow on affiliate links

### [ ] Link Depth
- Page accessible within 3 clicks from homepage
- Shallow site architecture
- **Pass:** ≤3 clicks from home
- **Fail:** >3 clicks, buried deep

## Media Optimization (7 items)

### [ ] Image Alt Text
- All images have descriptive alt text
- Primary keyword in main image alt (natural)
- Describes image content accurately
- **Pass:** All images have relevant alt text
- **Fail:** Missing alt or keyword stuffing

### [ ] Image File Names
- Descriptive file names (not IMG_1234.jpg)
- Lowercase, hyphen-separated
- Keyword-relevant
- **Pass:** "keyword-topic-image.jpg"
- **Fail:** "IMG_1234.jpg"

### [ ] Image Dimensions
- Proper width/height attributes set
- Prevents layout shift (CLS)
- Sized appropriately for display
- **Pass:** Dimensions match display size
- **Fail:** Oversized images scaled with CSS

### [ ] Video Optimization
- Video transcripts provided
- Video schema markup
- Thumbnail image optimized
- **Pass:** Transcript + schema present
- **Fail:** No transcript, missing schema

### [ ] Infographic Text Alternative
- Key data in HTML text (not just image)
- Screen reader accessible
- Text version of insights
- **Pass:** Information accessible in text
- **Fail:** Info only in image format

### [ ] PDF/Download Optimization
- Descriptive link text for downloads
- File size indicated
- Preview or description provided
- **Pass:** Clear link + file info
- **Fail:** Generic "Download PDF" link

### [ ] Favicon
- Favicon.ico in root directory
- Multiple sizes for different devices
- Brand-consistent icon
- **Pass:** Favicon displays in browser tabs
- **Fail:** Default browser icon or missing

## Additional Elements (3 items)

### [ ] Language Declaration
- Lang attribute in HTML tag
- Correct language code (en-US, es-MX)
- **Pass:** `<html lang="en">`
- **Fail:** Missing lang attribute

### [ ] Table of Contents
- TOC for content >2,000 words
- Jump links to sections
- Improves UX and featured snippets
- **Pass:** Interactive TOC present
- **Fail:** Long content without TOC

### [ ] FAQ Section
- FAQ schema markup
- Answers common user questions
- Natural language questions
- **Pass:** FAQ with schema implemented
- **Fail:** No FAQ or missing schema
