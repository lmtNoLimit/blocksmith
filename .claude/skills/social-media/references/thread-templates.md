# Thread Templates

## Thread Types

### 1. Educational Thread
```
Tweet 1 (Hook): Here's what I learned about [topic] after [timeframe/experience]

Tweet 2-N: [Key lessons, one per tweet]

Final Tweet: TL;DR + CTA
```

### 2. Story Thread
```
Tweet 1: The time I [experience]... (Thread)

Tweet 2: It started when [setup]

Tweet 3-N: [Story progression]

Final Tweet: The lesson + CTA
```

### 3. List Thread
```
Tweet 1: [Number] things every [audience] should know about [topic]:

Tweet 2-N: [One item per tweet with explanation]

Final Tweet: Summary + CTA
```

### 4. How-To Thread
```
Tweet 1: How to [achieve outcome] (step by step):

Tweet 2-N: Step [N]: [Action + brief explanation]

Final Tweet: Recap + CTA
```

### 5. Breakdown Thread
```
Tweet 1: Let me break down [complex topic] in simple terms:

Tweet 2-N: [Concept explanations]

Final Tweet: Summary + where to learn more
```

## Thread Best Practices

1. **Hook is everything** - First tweet determines success
2. **Keep tweets standalone** - Each should provide value
3. **Use visuals** - Add images/screenshots where relevant
4. **Number your tweets** - Makes following easier
5. **End with CTA** - Follow, retweet, comment, link

## Hook Formulas

- "I spent [time] studying [topic]. Here's what I found:"
- "Most people get [topic] wrong. Here's the truth:"
- "[Controversial statement]. Here's why:"
- "In [timeframe], I went from [state A] to [state B]. Here's how:"
- "[Number] [things] that [outcome]. (Thread)"

## Thread Length

- **Minimum:** 4 tweets (hook + 2 content + CTA)
- **Optimal:** 7-10 tweets
- **Maximum:** 15-20 tweets (beyond gets fewer completions)
